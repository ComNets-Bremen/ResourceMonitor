ResourceMonitor
===============

Introduction
------------

ResourceMonitor collects a variety of different phone usage parameters for
further analysis.

ResourceMonitor is available on [Google Play](https://play.google.com/store/apps/details?id=de.uni_bremen.comnets.resourcemonitor).
